# samf's Homebrew Tap

This is a custom Homebrew tap containing formulas for samf's projects.

## Installation

```bash
# Add the tap
brew tap samf/samf

# Install daypub
brew install daypub
```

## Available Formulas

### daypub
Convert Day One journal entries to markdown/Hugo format.

```bash
# Install
brew install samf/samf/daypub

# Usage
daypub --help
```
